# Digital Heroes Resume Builder

A modern ATS-friendly Resume Builder built using HTML, CSS, and JavaScript.

## Features

* Live Resume Preview
* Profile Photo Upload
* PDF Resume Download
* Dark Mode Support
* Skills Tags
* College & CGPA Section
* Certifications Section
* Local Storage Auto Save
* Mobile Responsive Design
* Professional Resume Layout

## Technologies Used

* HTML5
* CSS3
* JavaScript
* html2pdf.js

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
4. Fill in your details and generate your resume.
